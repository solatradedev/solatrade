# SolaTrade

Solana NFT Marketplace — buy, sell, and mint digital collectibles.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Environment Variables

Copy `.env.example` to `.env` and fill in:
- `VITE_SOLANA_RPC_URL` — Solana RPC endpoint
- `VITE_LMNFT_OWNER_ID` — LaunchMyNFT owner ID
- `VITE_LMNFT_COLLECTION_ID` — LaunchMyNFT collection ID
